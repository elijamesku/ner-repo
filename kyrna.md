# S3 Bucket Infrastructure

This infrastructure creates a simple S3 bucket with a random suffix to ensure global uniqueness. The bucket is configured with default settings and can be used for general-purpose object storage.

## Resources Created

- **aws_s3_bucket.main** - Primary S3 bucket
  - [Terraform Documentation](https://registry.terraform.io/providers/hashicorp/aws/latest/docs/resources/s3_bucket)
- **random_id.bucket_suffix** - Random suffix for unique bucket naming
  - [Terraform Documentation](https://registry.terraform.io/providers/hashicorp/random/latest/docs/resources/id)

## Deployment

```bash
# Initialize Terraform
terraform init

# Review planned changes
terraform plan

# Apply infrastructure
terraform apply
```

## Cost Estimate

**Monthly Cost**: ~$0.023 per GB stored + $0.0004 per 1,000 PUT requests + $0.0004 per 10,000 GET requests. Empty bucket incurs no storage costs.
